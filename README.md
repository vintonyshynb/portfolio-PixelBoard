# Interactive web application for sticky notes with drag & drop, color customization, and PHP/MySQL backend.

## Features
- Add, edit, delete notes
- Drag & drop notes on the board
- Change background and text color
- Stores notes in MySQL database

## Usage
1. Clone the repo
2. Setup a local server (XAMPP/MAMP)
3. Import `pixelboard.sql` to MySQL
4. Open `index.html` in browser
